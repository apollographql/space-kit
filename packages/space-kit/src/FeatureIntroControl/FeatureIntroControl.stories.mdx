# FeatureIntro

FeatureIntro is a set of components that are used as an introduction to a new component for a user.
FeatureIntroControl is a layout component that is a wrapper for FeatureIntroHeading, FeatureIntroContent, FeatureIntroImage, FeatureIntroLearnMoreLink and FeatureIntroDismissButton.

## Shortcut

All FeatureIntro components are exported from `FeatureIntro` with the preceeding `FeatureIntro` removed:

- `FeatureIntro.Control`
- `FeatureIntro.Heading`
- `FeatureIntro.Content`
- `FeatureIntro.Image`
- `FeatureIntro.LearnMoreLink`
- `FeatureIntro.DismissButton`
